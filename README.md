# 🏦 Banking Query Intent Classifier

An AI-powered web application that classifies banking customer 
queries into 77 intent categories using NLP and Machine Learning.

## 🔗 Live Demo
👉 [Click here to try the app](https://banking-intent-classifier-tumkuzyvxqgcbzwdmmufx4.streamlit.app)

## 📌 Project Overview
When customers contact bank support, agents manually read and 
route queries to the right department. This system automates 
that process using Machine Learning — saving time and improving 
response efficiency.

## 📊 Model Performance
| Metric | Value |
|--------|-------|
| Accuracy | 85.62% |
| Total Intents | 77 |
| Training Samples | 10,003 |
| Testing Samples | 3,080 |

## 🛠️ Tech Stack
- **Language:** Python
- **ML Model:** Logistic Regression
- **Text Processing:** TF-IDF Vectorization
- **Libraries:** Scikit-learn, Pandas, NumPy, Matplotlib
- **Web App:** Streamlit
- **Dataset:** Banking77 (PolyAI)

## 🚀 Features
- 🔐 Customer login page with Bank ID verification
- 💬 Real-time intent prediction
- 📊 Top 5 intent confidence chart
- 🕐 Recent query history
- 🟢 Confidence level indicator (High/Medium/Low)
